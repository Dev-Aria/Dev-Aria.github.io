1. Bootstrap이란 무엇인가?

Bootstrap : Front-End 오픈소스 프레임워크

  • 효율적인 웹 디자인 을 위해 미리 필요한 부분을 작업 해 놓은 것
  • HTML, CSS, JavaScript 및 다양한 UI 컴포넌트로 구성된 오픈소스 프레임워크
  • 2010년 중반에 트위터 개발자 중 @mdo와 @fat이 개발한 오픈소스 프레임워크
  • 깃허브 에서 호스팅, 개발, 유지보수
  • 링크 : http://bootstrapk.com/

3. eGovFramework 프로젝트에 Bootstrap 적용

  • 프로젝트의 webapp 폴더에 audio, css, fonts, images, js 덮어씌우기


  • 구현할 페이지 선정 후, F12를 이용해 레이아웃 확인

    • 구현할 페이지 : blogone.html


  • 구현할 페이지의 레이아웃을 tiles에 적용

    • 이클립스에서 blogone.html 파일을 열어보면 소스코드를 확인할 수 있다.
    • 레이아웃 별로 똑같이 복사해 서버에 돌려본 후 제대로 구현된 것을 확인한 후 커스터마이징 한다.

    • 파일경로 : WEB-INF/jsp/egovframework/example/cmmn/파일명.jsp

    • default-layouts.jsp


  • default-header.jsp


  • default-nav.jsp


  • default-pageTitle.jsp


  • default-rightBar.jsp


  • default-footer.jsp


  • tiles의 content에 해당하는 main.jsp에 코드 저장

    • 파일위치 : WEB-INF/jsp/egovframework/example/main/main.jsp
    • main.jsp


  • 불필요한 부분은 제거해 커스터마이징 하기

    • 수정한 파일 : main.jsp / main.css / default-pageTitle.jsp / default-rightBar.jsp / default-footer.jsp
