04-2. MySQL Workbench에 SChema와 Table 생성

2. MySQL Workbench에서 새로운 연결 생성
3. 실행 중인 MySQL과 연결
4. 새로운 Model 생성 후, Table 추가
5. 설계한 Model을 실제 DB에 반영
6. 실제 DB에 반영됐는지 조회
7. 기본 SQL 문법





1.MySQL와 Workbench 다운로드 및 설치

  • 다운로드 경로 : https://www.mysql.com/
  • MySQL Community Edition 다운로드
  • PC 환경에 맞는 OS 선택 후 다운로드 (Windows)

  • 압축해제 후 CMD로 설치 진행


  • root계정 패스워드 변경(반드시 use mysql 후 변경)


  • 패스워드 변경 후, flush privileges; (반드시 해야 변경사항 적용됨)



2. MySQL Workbench에서 새로운 연결 생성

  • Workbench 실행 후 Connection 옆의 + 버튼을 클릭


  • Connection Name, Hostname, Username 입력 후, Test Connection



3. 실행 중인 MySQL과 연결

  • MySQL Connections에 추가된 Project_Basic 에 접속


  • 접속에 성공한 화면



4. 새로운 Model 생성 후, Table 추가

  • File > New Model / Ctrl+N


  • mydb을 더블클릭하여 이름 변경


  • 생성된 Model Project_Basic에 Table 넣기


  • 설계한 Table(Excel)



5. 설계한 Model을 실제 DB에 반영

  • Database > Forward Engineer 클릭

    • Forward Engineer: 자신의 설계한 Model을 실제 DB에 반영하는 것

    • Reverse Engineer: DB의 내용을 가지고 Model을 만드는 것















6. 실제 DB에 반영됐는지 조회

  • MySQL Model을 닫고 Local instance로 복귀
  • refresh
  • 해당 Schema를 기본으로 설정
    • 우클릭 - Set as Default Schema
    • 해당 Schema가 진하게 표시된 것 확인





  • CMD 창에서도 확인



7. 기본 SQL 문법

  • 설계한 columns의 내용 확인
    • desc 테이블명; 입력 후 실행 (번개모양)


  • table에 data 삽입 후 columns 확인
    • insert into table명 (column1, column2, column3) valuse (value1, value2, value3);
    • select * from table명;
