04. eGovFrame 프로젝트에 DB 연동하기 (3)

04-3. Ecilpse에 MySQL 연동

1. mysql-connector 설치
2. eclipse 실행 후, Data Source Explorer 창 열기
3. MySQL 연동하기
4. DB가 정상적으로 연결됐는지 확인하기





1. mysql-connector 설치

  • 다운받은 파일을 압축해제 한 후, connector.jar 파일을 복사해 mysql 설치 폴더로 이동



2. eclipse 실행 후, Data Source Explorer 창 열기

  • Window탭 - Show View - Other - Data Management - Data Soruce Explorer



3. MySQL 연동하기

  • New Connection Profile 클릭 - MySQL 선택 - Name 지정


  • New Driver Definition 클릭 - MySQL JDBC 5.1 Version 선택 후 JAR List 클릭


  • 기존의 jar 파일 Remove JAR/Zip 후, Add JAR/Zip 클릭

    • 4-1에서 MySQL 폴더에 복사해 둔 jar 파일을 선택


  • default로 저장된 정보를 커스터마이징 하기

    • Database - 지정한 이름, password는 MySQL 설치 시 지정한 비밀번호
    • 입력 후, Text Connection 클릭
    • Ping succeeded! 확인하기
      • failed가 뜰 경우, 비밀번호 확인 / MySQL의 실행 여부 확인(mysqld.exe)


  • 설정 내역 확인 후 Finish 클릭



4. DB가 정상적으로 연결됐는지 확인하기

  • Data source Explorer에 연결된 MySQL Schema와 테이블 확인


  • 이클립스 재접속 시 따로 Connection 하기. (MySQL 실행 후)
