03. CSS로 웹 페이지를 예쁘게 꾸미자

03-1. CSS 작성 방법

1. Inline 방식 바로가기
2. Internal 방식 바로가기
3. External 방식 바로가기



CSS 란?

  • Cascading Style Sheets의 약자
  • HTML 요소들의 색상, 위치, 크기 등을 제어하는 스크립트 언어
  • HTML 뼈대에 스타일을 적용하기 위한 코드를 추가하는 형태로 작성

CSS 작성 방법

  • Inline, Internal, External
  • 각각의 작성법이나 스타일의 적용 범위가 다름


1. Inline 방식
  • 스타일을 적용할 태그 에 style 속성으로 설정
  • 태그가 끝날 때 까지만 유효

  • 주로 스타일을 한 번만 적용할 때 사용

  • Inline 기본형




2. Internal 방식
  • HTML의 head태그 안 에 style태그로 스타일 정의

  • 현재 웹 페이지 내의 요소들을 태그/id/class 별로 스타일을 공통 적용할 때 사용하는 방법

  • Internal 기본형 태그명

    • 현재 페이지 내 의 동일한 태그에 공통으로 스타일 적용

  • Internal 기본형 id

    • 태그명에 관계없이 현재 페이지 내 의 id에 스타일 적용

  • Internal 기본형 class

    • 태그명에 관계없이 현재 페이지 내 의 class에 스타일 적용


  • Internal 실습 예제




3. External 방식

  • Internal 방식과 동일하게 적용한 CSS 코드를 외부 파일에 저장해 HTML에서 링크

  • 다수의 페이지에서 동일한 style을 적용할 때 사용

  • link 태그

    • 외부의 문서를 연결하는 태그
    • 사용법
      • head 태그 내부 에 위치
      • 여러개 의 link태그 삽입 가능
    • 속성
      • href: 연결할 곳의 주소(절대주소, 상대주소)
      • rel: 현재문서와 연결문서의 관계 표시
        • stylesheet: 스타일시트로 연결할 때
        • alternate: 문서의 대안 버전으로 연결할 때(프린트, 번역사이트)
        • author: 저작자로 연결할 때
        • help: 도움말오 연결할 때
        • license: 문서의 저작권 정보로 연결할 때
        • search: 검색 도구로 연결할 때
        • icon: 파비콘 관계의 연결
      • media: 연결 문서가 표시될 장치 또는 미디어 유형
        • all: 기본값. 모든 장치에 알맞음
        • tv: 텔레비전 유형의 장치
        • screen: 컴퓨터 화면, 태블릿, 스마트폰 화면
        • print: 프린트 미리보기와 인쇄 페이지 모드
        • speech: 페이지를 읽어주는 스크린 리더
