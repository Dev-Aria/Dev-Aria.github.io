1. 브라우저 객체란?

  • 브라우저 객체 모델(BOM, Browser Object Model)

    • 계층적 구조 로 설계된 브라우저 객체

    • 브라우저 객체: 브라우저에 내장된 객체

      • window
        • Browser 객체의 최상위 객체
        • 하위 객체 포함
          • document, screen, location, history, navigator




2. window 객체

브라우저 객체의 최상위 객체

  • window 객체의 메서드
    • open(), alert(), prompt(), confirm(), moveTo(), resizeTo(), setInterval(), setTimeout()



window.open()

  • 팝업 창
  • 지정한 URL 페이지를 새 브라우저 창에 나타내기
  • 팝업창 제작에 용이
    • 광고에 사용 가능하나, 팝업 차단이 가능하므로 사용하지 않는 추세
  • open() 메서드의 옵션
    • width, height, left, top, scrollbars, location, status, toolbars

  • window.open() 기본형



  • window.open() 예제



alert()

  • 경고 창
  • window 객체를 따로 작성하지 않아도 사용 가능

  • alert() 기본형

    • window.alert()와 alert()의 기능이 같은 것 확인



prompt()

  • 질의응답 창
  • window 객체를 따로 작성하지 않아도 사용 가능
  • [확인] 클릭 시, 입력한 답변 반환

  • prompt() 기본형

    • window.prompt()와 prompt()의 기능이 같은 것 확인



confirm()

  • 확인/취소 창
  • window 객체를 따로 작성하지 않아도 사용 가능
  • [확인] 클릭 시, true 반환
  • [취소] 클릭 시, false 반환

  • confirm() 기본형

    • window.confirm()와 confirm()의 기능이 같은 것 확인




3. 일정한 시간 간격으로 코드 실행하기
  • setInterval()
    • 일정한 시간 간격으로 코드 반복 실행
  • setTimeout()
    • 일정한 시간 후에 코드 실행 후 종료
  • 차이점
    • setInterval(): 3초 간격으로 코드 반복 실행
    • setTimeout(): 3초 후에 코드를 한 번만 실행 후 종료



setInterval()/ clearInterval()

  • setInterval()
    • 일정한 시간 간격으로 코드 반복 실행
  • clearInterval()
    • setInterval() 메서드 종료

  • setInterval()/ clearInterval() 기본형

    • 시간 간격: 1/1,000초 단위(ms)
      • 3초 간격 = 3,000
    • 참조변수: clearInterval() 메서드에 사용

  • setInterval()/ clearInterval() 예제
