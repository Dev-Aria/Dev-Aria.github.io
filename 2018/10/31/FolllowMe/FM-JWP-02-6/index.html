form 태그의 개념
  • 뷰 페이지
    • 사용자에게 요청의 결과 보여주기
    • 사용자의 입력 받기
  • 폼(form) 태그
    • 사용자가 입력한 데이터를 수집해 서버로 전송하는 역할
    • 입력 양식들을 하나의 단위로 묶어 서버로 전송
      • 여러 종류의 입력 양식을 사용 가능
    • 속성
      • action : form 태그가 전송한 데이터를 받을 서버 페이지의 위치 지정
      • method : 데이터를 서버로 전송하는 방식 지정
        • get방식
          • 입력 데이터를 url에 연결하여 전송하므로 웹 페이지 주소와 함께 그대로 노출
          • 보안에 민감한 데이터를 get으로 전송하면 좋지 않음
        • post방식
          • 패킷 안에 데이터를 담아 전송하기 때문에 정보가 외부에 보이지 않음
          • 보안에 민감한 데이터를 전송할 때 사용
    • input 태그
      • 속성
        • type : input 태그의 종류 지정
          • text, checkbox, radio, hidden, button, reset 등
        • name : 현재 태그의 이름
        • value : 입력 양식의 값
          • 읽기, 입력 가능
          • 값 할당 : 사용자가 입력 하지 않아도 입력 양식에 디폴트 출력
          • 사용자 입력 : value 속성으로 읽어올 수 있음

1. 텍스트 박스
  • input : type : text
  • 한 줄 입력 박스

  • 속성

    • disabled : 해당 입력 양식 사용 불가
    • readonly : 디폴트 문자열 출력, 입력 값 수정 불가
  • 텍스트 박스 실습




2. 패스워드 박스
  • input : type : password
  • 텍스트 박스와 동일

  • 차이 : 입력 시에 데이터가 보이지 않음

  • 패스워드 박스 실습




3. 체크 박스
  • input : type : checkbox
  • 리스트 중에서 선호하는 것을 선택하는 입력 양식
    • 다중 선택 가능

  • 속성

    • name
      • 동일하게 설정
      • 여러 개의 체크 박스를 하나의 그룹으로 묶기
    • value
      • 해당 체크박스의 값 = 서버로 전송되는 값 ≠ 화면에 출력되는 값

  • 체크 박스 실습




4. 라디오 버튼
  • input : type : radio
  • 한 그룹의 리스트에서 하나만 선택할 수 있는 입력 양식
    • 단일 선택 가능

  • 속성

    • name
      • 동일하게 설정
      • 여러 개의 체크 박스를 하나의 그룹으로 묶기
    • value
      • 해당 체크박스의 값 = 서버로 전송되는 값 ≠ 화면에 출력되는 값

  • 라디오 버튼 실습




5. 텍스트 영역
  • input 태그 text : 한 줄 입력 가능
  • textarea 태그 : 여러 줄 입력 가능

  • 속성

    • rows : 텍스트 영역의 세로 길이 설정 : 줄 수
    • cols : 텍스트 영역의 가로 길이 설정 : 글자 수
    • style : resize : 텍스트 영역 크기 조절
      • none : 높이, 너비 조절 불가
      • vertical : 높이 조절
      • horizontal : 너비 조절
      • both : 높이, 너비 조절 모두 가능

  • 텍스트 영역 실습




6. 콤보 박스
  • select 태그 : 목록을 드롭 다운(Drop-Down)으로 보여줌
    • select: 전체 목록
    • option: 목록의 항목

  • 선택한 항목의 value 값이 서버로 전송

  • 콤보 박스 실습




7. 히든

  • input : type : hidden

    • 웹 페이지 실행 시, 화면에는 보이지 않지만 서버로 전송되는 value 값

  • 히든 실습




8. 일반 버튼

  • input : type : button

    • value : 버튼 표면의 텍스트

  • 일반 버튼 실습




9. 전송 버튼

  • input : type : submit

    • 사용자가 입력 양식에 입력한 데이터를 서버로 전송
    • 반드시 submit 버튼을 클릭해야 서버로 전송됨
    • 속성
      • action : 지정한 서버 페이지로 데이터 전송
      • method : 전송 방식 설정
      • value : 버튼 표면에 출력되는 텍스트

  • 전송 버튼 실습




10. 취소 버튼

  • input : type : reset

    • 입력 양식에 입력한 데이터 초기화

  • 취소 버튼 실습
