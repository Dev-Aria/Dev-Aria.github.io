02-5. 테이블

1. 테이블

  • 표 출력
    • table: 테이블 정의
    • tr = 행, 줄 = row
    • td = 열, 칸 = column
  • 화면의 레이아웃 제작
    • 화면 분할

  • style

    • 테두리두께: border: 미지정시 0
    • 크기: width, height: 미지정시 내용크기 만큼

  • table 기본형




2. 테이블에 제목 줄과 캡션 추가하기

  • 제목 출력: th 태그: 진한 글씨

  • 캡션 달기: caption 태그

  • table 제목, 캡션




3. 칸 합치기(셀 병합)

  • 가로: td: colspan=”합칠 칸의 수”

  • 세로: td: rowspan=”합칠 칸의 수”

  • table 셀 병합




4. 테이블에 이미지 출력

  • 가로: td: colspan=”합칠 칸의 수”

  • 세로: td: rowspan=”합칠 칸의 수”

  • table 이미지 출력
