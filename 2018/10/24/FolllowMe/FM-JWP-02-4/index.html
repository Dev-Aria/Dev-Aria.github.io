02-4. 페이지 이동과 이미지 출력, 리스트 출력

1. 페이지 이동

  • a 태그 로 구현

    • 하이퍼링크(Hyperlink)
      • 텍스트나 이미지를 클릭해 다른 페이지로 이동하는 것
    • 타겟(target)
      • 링크 페이지를 어느 윈도우에 출력할지 설정
    • 아이디(id)
      • 북마크(Bookmark) 생성
      • 페이지 내용이 많아 스크롤이 길게 생겼을 경우,
        페이지 내에서 원하는 곳으로 이동할 수 있게 표시하는 것
      • 다른 페이지로도 이동 가능

  • a 태그 속성 기본형


  • 페이지 이동 샘플 코드1 - main

  • 페이지 이동 샘플 코드2 - target, bookmark

  • 페이지 이동 샘플 코드3 - bookmark











2. 경로

  • 경로와 파일명이 실제 dir와 일치 해야 함
    • 다른 페이지로 이동
    • 이미지 출력
  • 동일한 폴더
    • 파일명 명시
  • 다른 폴더
    • 경로 + 파일명

  • 경로 표기법

    • 절대 경로
      • 루트로부터 파일이 있는 위치까지의 전체 경로 모두 작성
    • 상대 경로
      • 현재 파일(작성하는 웹 페이지)을 기준으로 자원에 접근하는 경로
      • 상위 폴더: ../
      • 하위 폴더: 폴더명

  • EX. ch1\1 에서 ch2\2 로 링크걸기

    • C:\data\source\ch1\1.html
    • C:\data\source\ch2\2.html

    • 절대경로

    • 상대경로



3. 이미지 출력

  • img 태그

    • body 없이 속성만 가짐
      • src : 출력하려는 이미지 경로
      • alt : 이미지가 출력되지 않을 경우 보여줄 텍스트
      • style
        • 크기 조절
          • 크기 미지정시, 원본 이미지 크기로 출력
          • width: 가로, height: 세로
          • 단위 - px(픽셀), pt(포인트)
        • 정렬
          • float- right(오른쪽), left(왼쪽)

    • 이미지에 링크 설정 가능

    • img 태그 기본형

  • HTML Img 샘플 코드



4. 리스트 출력

  • 웹 페이지에 목록 출력
    • 순서 O
      • ul 태그 (style=”list-style-type”)
        • disc: 안이 채워진 원 모양(기본값)
        • circle: 안이 빈 원 모양
        • square: 사각형 모양
        • none: 아무 마크도 표시되지 않음
    • 순서 X
      • ol 태그 (type=)
        • type = “1” start=”3” : 3, 4, 5
        • type = “A” start=”4” : D, E, F
        • type = “a” start=”2” : b, c, d
        • type = “I” start=”2” : II, III, IV
        • type = “i” start=”2” : ii, iii, iv
